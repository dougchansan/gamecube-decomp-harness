7 · Winning-tactic taxonomy

The 16 gate-passing exact summaries cluster into six recurring MWCC-shaped moves:

  • Register-allocation steering via locals (5 wins) — introduce, reorder, or remove pointer/scalar temporaries so MWCC assigns r28–r30 as the target expects (fn_80175D34, fn_80174284, mnDiagram_OnFrame, ftColl_8007925C, un_803148E4).
  • Static-data layout reconstruction (4 wins) — adjacent-static base selection, contiguous data tables with original padding, moving .sdata2 constant definitions (gm_8016A164, mpLib_DrawSpecialPoints, ftColl_80077970, fn_80186EFC).
  • Type narrowing / bitfield typing (2 wins) — s32→u8 temporary, typed bitfield access instead of flag arithmetic (gm_80164910, fn_8016719C).
  • Natural-shape restoration (2 wins) — replace generated-looking unrolled/manual code with the obvious loop or array expression (grCastle_801CD658, AXDriver_8038E498).
  • Inline-helper reuse from analogous functions (1 win) — reuse existing inlined count helpers from the sibling fog callback (fn_801A94BC).
  • Declaration-order-only reshuffles (1 win) — pure decl reorder, zero logic change (fn_80174284).

8 · Where confirmations are being lost

  • 3 exact matches died at the lint gate for the same violation: no-string-literal-symbol-regression — the agent replaced an inline string literal with a data-label reference (fn_80393C14, mnCount_Create, grKongo_801D7BBC). One sentence in the worker prompt ("never replace string literals with data labels") likely converts these directly into confirmations.
  • 2 lost at runner validation — one caused 10 same-unit regressions, one didn't reproduce outside the worktree. Both medium-thinking sessions; checkdiff_summary on neighbors before reporting catches the first class, and 96% of exact winners already run it.
  • 159 aborted xhigh sessions (~22 min mean each ≈ 60 worker-hours) burned on empty model returns — consistent with the known LB-outage failure mode. Retry-on-empty-return would recover the single biggest throughput loss in this run.
  • All 8 regressions came from medium-thinking sessions. If medium stays in the mix for cost reasons, gate its returns harder; xhigh produced zero.
Method & caveats.
