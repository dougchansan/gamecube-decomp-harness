# Discord Knowledge Indexes

Planned source-local exports:

- `chunks.jsonl`
- `vector.sqlite`
- `vector_manifest.json`
- `topics.jsonl`
- `terms.jsonl`
- `citations.jsonl`
- `file_mentions.jsonl`

`chunks.jsonl` is the canonical generated text chunk index. `vector.sqlite` and
`vector_manifest.json` are generated by `commands/vectorize.py` for semantic
RAG lookup. Until deeper topic/term exports exist, this slice remains a
registered optional source and does not affect graph ranking.
