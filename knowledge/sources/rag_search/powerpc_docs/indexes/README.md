# PowerPC Documents Indexes

Planned source-local exports:

- `pages.jsonl`
- `vector.sqlite`
- `vector_manifest.json`
- `instruction_terms.jsonl`
- `abi_terms.jsonl`
- `citations.jsonl`

The existing CSV page index at `data/indexes/powerpc_pdf_pages.csv` is the
current source of searchable text. `vector.sqlite` and
`vector_manifest.json` are generated by `commands/vectorize.py` for semantic
RAG lookup.
