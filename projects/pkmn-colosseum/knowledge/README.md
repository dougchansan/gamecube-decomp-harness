# Pokemon Colosseum Knowledge

Project-owned generated graph indexes, tool caches, and curated knowledge for
Pokemon Colosseum live under this directory. Generated graph databases,
knowledge indexes, session state, and tool output are ignored by the root
`.gitignore`.

The initial tracked source registry keeps the project focused on current
`GC6E01` code graph evidence while leaving room for Colosseum-specific corpora
to be added later.
