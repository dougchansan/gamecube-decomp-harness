# Agent Code Quality Standard Examples

This directory is reserved for standard-linked examples used by repair,
pre-ship review, and PR fixer prompts. Worker prompts should receive compact
standards summaries, not the full example catalog.

Expected example records should be lookupable by `standard_id` and optionally
`qa_rule_id`, and should show a small bad pattern, preferred source shape, and
description bullets explaining why the repair matters.
